# RevOps Assistant

Aplicación de escritorio (Electron + React + TypeScript) para operaciones de RevOps sobre HubSpot y Google Drive, con identidad de marca Cloud District.

Scaffolding generado en **SPEC-0001 — Fundación del Proyecto**.

## Inicio rápido

```cmd
pnpm install
pnpm dev
```

## Documentación

- `doc/instalacion.md` — instalación, scripts y entorno.
- `doc/arquitectura.md` — procesos, seguridad, IPC y tema.
- `specs/` — SPECs del proyecto (índice en `CLAUDE.md`).

## Stack

Electron · TypeScript (strict) · React 18 · MUI v5 · Zustand · i18next · electron-vite · electron-builder · Vitest · Playwright + axe-core.

## Idiomas

Castellano (por defecto), catalán, euskera e inglés. Traducciones en `src/renderer/locales/`.
