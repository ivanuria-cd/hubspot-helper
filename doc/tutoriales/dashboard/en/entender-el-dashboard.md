# Understanding the Dashboard

**Prerequisites:** an open project.
**Estimated time:** 2 minutes

The Dashboard is the first screen when you open a project. It summarizes, at a glance, what you have connected and what changes you still need to apply in HubSpot.

## Steps

1. Open a project from the welcome screen. You'll see the Dashboard.
2. In **Connectors** review the status of HubSpot, Google Drive and API / MCP. A green badge means it is connected; a neutral badge means it is not configured yet.
3. Click **Configure** on any card to go to that connector's screen.
4. In **Pending changes** you'll see how many unapplied changes you have in Properties, Custom objects and Forms.
5. Click **Review** to go to that area and apply or discard the changes. If it says **Up to date**, there is nothing pending.

## Expected result

You know in seconds whether your connectors are ready and where you have pending work, without going screen by screen.

## FAQ

**I don't see the cards, only "Getting started".** This is normal if you haven't configured any connector yet: follow the links to connect them.

**The counters don't add up.** Use the reload button in the header to query the portal status again.
