# Dashboard ulertu

**Aurrebaldintzak:** proiektu bat irekita izatea.
**Estimatutako denbora:** 2 minutu

Dashboard-a proiektu bat irekitzean agertzen den lehen pantaila da. Begirada bakar batean laburbiltzen du zer duzun konektatuta eta zein aldaketa geratzen zaizkizun HubSpot-en aplikatzeke.

## Urratsak

1. Ireki proiektu bat ongietorri-pantailatik. Dashboard-a ikusiko duzu.
2. **Konektoreak** atalean berrikusi HubSpot, Google Drive eta API / MCP-ren egoera. Etiketa berde batek konektatuta dagoela adierazten du; etiketa neutro batek, oraindik konfiguratu gabe dagoela.
3. Sakatu **Konfiguratu** edozein txarteletan konektore horren pantailara joateko.
4. **Aldaketa zain** atalean ikusiko duzu zenbat aldaketa dituzun aplikatzeke Propietateak, Objektu custom-ak eta Formularioak ataletan.
5. Sakatu **Berrikusi** arlo horretara joan eta aldaketak aplikatu edo baztertzeko. **Eguneratuta** jartzen badu, ez dago ezer zain.

## Espero den emaitza

Segundo gutxitan dakizu zure konektoreak prest dauden eta non duzun lana zain, pantailaz pantaila sartu gabe.

## Ohiko galderak

**Ez ditut txartelak ikusten, «Lehen urratsak» bakarrik.** Normala da oraindik konektorerik konfiguratu ez baduzu: jarraitu esteken bidez haiek konektatzeko.

**Kontagailuek ez dute bat egiten.** Erabili goiburuko birkargatze-botoia atariaren egoera berriz kontsultatzeko.
