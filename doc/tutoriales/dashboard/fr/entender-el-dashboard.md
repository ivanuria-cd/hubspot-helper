# Comprendre le Tableau de bord

**Prérequis :** avoir un projet ouvert.
**Temps estimé :** 2 minutes

Le Tableau de bord est le premier écran qui s'affiche à l'ouverture d'un projet. Il résume, d'un coup d'œil, ce que vous avez connecté et les modifications qu'il vous reste à appliquer dans HubSpot.

## Étapes

1. Ouvrez un projet depuis l'écran d'accueil. Vous verrez le Tableau de bord.
2. Dans **Connecteurs**, vérifiez l'état de HubSpot, Google Drive et API / MCP. Une étiquette verte indique qu'il est connecté ; une étiquette neutre, qu'il n'est pas encore configuré.
3. Cliquez sur **Configurer** sur n'importe quelle carte pour accéder à l'écran de ce connecteur.
4. Dans **Modifications en attente**, vous verrez combien de modifications vous avez sans appliquer dans Propriétés, Objets personnalisés et Formulaires.
5. Cliquez sur **Examiner** pour accéder à ce domaine et appliquer ou rejeter les modifications. S'il est indiqué **À jour**, il n'y a rien en attente.

## Résultat attendu

Vous savez en quelques secondes si vos connecteurs sont prêts et où il vous reste du travail en attente, sans entrer écran par écran.

## Questions fréquentes

**Je ne vois pas les cartes, seulement « Premiers pas ».** C'est normal si vous n'avez encore configuré aucun connecteur : suivez les liens pour les connecter.

**Les compteurs ne correspondent pas.** Utilisez le bouton de rechargement de l'en-tête pour consulter à nouveau l'état du portail.
