# Compreender o Dashboard

**Pré-requisitos:** ter um projeto aberto.
**Tempo estimado:** 2 minutos

O Dashboard é o primeiro ecrã ao abrir um projeto. Resume, num relance, o que tem ligado e que alterações lhe faltam aplicar no HubSpot.

## Passos

1. Abra um projeto a partir do ecrã de boas-vindas. Verá o Dashboard.
2. Em **Conetores**, reveja o estado do HubSpot, do Google Drive e da API / MCP. Uma etiqueta verde indica que está ligado; uma etiqueta neutra, que ainda não está configurado.
3. Prima **Configurar** em qualquer cartão para ir para o ecrã desse conetor.
4. Em **Alterações pendentes** verá quantas alterações tem por aplicar em Propriedades, Objetos custom e Formulários.
5. Prima **Rever** para ir para essa área e aplicar ou descartar as alterações. Se indicar **Atualizado**, não há nada pendente.

## Resultado esperado

Sabe em segundos se os seus conetores estão prontos e onde tem trabalho pendente, sem entrar ecrã a ecrã.

## Perguntas frequentes

**Não vejo os cartões, apenas «Primeiros passos».** É normal se ainda não tiver configurado nenhum conetor: siga as ligações para os ligar.

**Os contadores não batem certo.** Use o botão de recarregar do cabeçalho para voltar a consultar o estado do portal.
