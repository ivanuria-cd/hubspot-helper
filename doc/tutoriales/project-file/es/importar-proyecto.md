# Importar un proyecto desde un archivo

**Prerrequisitos:** tener un archivo `.rvproj` exportado previamente.
**Tiempo estimado:** 2 minutos

Puedes recrear un proyecto en tu equipo a partir de un archivo `.rvproj`.

## Pasos

1. En la pantalla de bienvenida, pulsa **Importar proyecto**.
2. Selecciona el archivo `.rvproj` y confirma.
3. Revisa el **resumen previo**: nombre, conectores referenciados y contenido que se importará.
4. Si ya existe un proyecto con el mismo identificador, elige **Crear copia** (opción por defecto) o **Sobrescribir el existente**.
5. Pulsa **Importar**.
6. Tras importar, reconecta **HubSpot** y **Google Drive** con tus credenciales.

## Resultado esperado

El proyecto aparece en la lista y queda abierto, con su mapa de propiedades, objetos custom y formularios.

## Preguntas frecuentes

**¿Por qué tengo que reconectar los conectores?** Porque el archivo no contiene credenciales por seguridad; solo las referencias al portal y a la carpeta.

**¿Qué pasa si una sección es de una versión más nueva que mi app?** Se omite al importar pero se conserva en el archivo, de modo que no se pierde si vuelves a exportar.

**¿Y si el archivo está alterado?** Si la comprobación de integridad no cuadra, verás un aviso en el resumen antes de importar.
