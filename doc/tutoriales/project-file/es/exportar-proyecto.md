# Exportar un proyecto a un archivo

**Prerrequisitos:** tener un proyecto en la lista de bienvenida.
**Tiempo estimado:** 1 minuto

Puedes guardar un proyecto completo en un único archivo `.rvproj` para compartirlo o trasladarlo a otro equipo.

## Pasos

1. En la pantalla de bienvenida, localiza el proyecto en la lista.
2. Pulsa el icono **Exportar** (flecha hacia abajo) en la fila del proyecto.
3. Elige dónde guardar el archivo `.rvproj` y confirma.
4. Verás un aviso de confirmación con la ruta donde se ha guardado.

## Resultado esperado

Un archivo `.rvproj` con el mapa de propiedades, los objetos custom y los formularios del proyecto, listo para compartir.

## Preguntas frecuentes

**¿Incluye mis credenciales?** No. El archivo guarda solo referencias (portal de HubSpot, carpeta de Drive), nunca tokens ni contraseñas. Quien lo importe deberá reconectar sus propios conectores.

**¿Qué contiene exactamente?** Los metadatos del proyecto y el estado de cada característica del CRM. No incluye los datos vivos de HubSpot, solo la configuración que gestiona la app.
