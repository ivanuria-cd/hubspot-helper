# Exporter un projet vers un fichier

**Prérequis :** avoir un projet dans la liste d'accueil.
**Temps estimé :** 1 minute

Vous pouvez enregistrer un projet complet dans un unique fichier `.rvproj` pour le partager ou le transférer sur une autre machine.

## Étapes

1. Sur l'écran d'accueil, repérez le projet dans la liste.
2. Cliquez sur l'icône **Exporter** (flèche vers le bas) sur la ligne du projet.
3. Choisissez où enregistrer le fichier `.rvproj` et confirmez.
4. Vous verrez un avis de confirmation indiquant le chemin où il a été enregistré.

## Résultat attendu

Un fichier `.rvproj` contenant la carte des propriétés, les objets personnalisés et les formulaires du projet, prêt à être partagé.

## Questions fréquentes

**Inclut-il mes identifiants ?** Non. Le fichier n'enregistre que des références (portail HubSpot, dossier Drive), jamais de jetons ni de mots de passe. Celui qui l'importe devra reconnecter ses propres connecteurs.

**Que contient-il exactement ?** Les métadonnées du projet et l'état de chaque fonctionnalité du CRM. Il n'inclut pas les données vivantes de HubSpot, seulement la configuration que gère l'application.
