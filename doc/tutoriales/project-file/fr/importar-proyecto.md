# Importer un projet depuis un fichier

**Prérequis :** avoir un fichier `.rvproj` exporté au préalable.
**Temps estimé :** 2 minutes

Vous pouvez recréer un projet sur votre machine à partir d'un fichier `.rvproj`.

## Étapes

1. Sur l'écran d'accueil, cliquez sur **Importer un projet**.
2. Sélectionnez le fichier `.rvproj` et confirmez.
3. Examinez le **résumé préalable** : nom, connecteurs référencés et contenu qui sera importé.
4. S'il existe déjà un projet avec le même identifiant, choisissez **Créer une copie** (option par défaut) ou **Écraser l'existant**.
5. Cliquez sur **Importer**.
6. Après l'importation, reconnectez **HubSpot** et **Google Drive** avec vos identifiants.

## Résultat attendu

Le projet apparaît dans la liste et reste ouvert, avec sa carte des propriétés, ses objets personnalisés et ses formulaires.

## Questions fréquentes

**Pourquoi dois-je reconnecter les connecteurs ?** Parce que le fichier ne contient pas d'identifiants pour des raisons de sécurité ; seulement les références au portail et au dossier.

**Que se passe-t-il si une section provient d'une version plus récente que mon application ?** Elle est ignorée à l'importation mais conservée dans le fichier, de sorte qu'elle n'est pas perdue si vous réexportez.

**Et si le fichier est altéré ?** Si la vérification d'intégrité ne correspond pas, vous verrez un avertissement dans le résumé avant l'importation.
