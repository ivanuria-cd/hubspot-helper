# Export a project to a file

**Prerequisites:** have a project in the welcome list.
**Estimated time:** 1 minute

You can save a complete project to a single `.rvproj` file to share it or move it to another machine.

## Steps

1. On the welcome screen, find the project in the list.
2. Click the **Export** icon (down arrow) on the project row.
3. Choose where to save the `.rvproj` file and confirm.
4. A confirmation notice shows the path where it was saved.

## Expected result

A `.rvproj` file with the project's property map, custom objects and forms, ready to share.

## FAQ

**Does it include my credentials?** No. The file stores only references (HubSpot portal, Drive folder), never tokens or passwords. Whoever imports it must reconnect their own connectors.

**What exactly does it contain?** The project metadata and the state of each CRM feature. It does not include live HubSpot data, only the configuration the app manages.
