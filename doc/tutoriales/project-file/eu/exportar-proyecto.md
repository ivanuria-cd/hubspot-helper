# Esportatu proiektu bat fitxategi batera

**Aurrebaldintzak:** proiektu bat izatea ongietorri-zerrendan.
**Denbora estimatua:** minutu 1

Proiektu oso bat `.rvproj` fitxategi bakar batean gorde dezakezu, partekatzeko edo beste ekipo batera eramateko.

## Urratsak

1. Ongietorri-pantailan, aurkitu proiektua zerrendan.
2. Sakatu **Esportatu** ikonoa (behera gezia) proiektuaren errenkadan.
3. Aukeratu `.rvproj` fitxategia non gorde eta berretsi.
4. Gorde den bidea adierazten duen berrespen-oharra ikusiko duzu.

## Espero den emaitza

Proiektuaren propietate-mapa, objektu pertsonalizatuak eta inprimakiak dituen `.rvproj` fitxategi bat, partekatzeko prest.

## Ohiko galderak

**Nire kredentzialak biltzen ditu?** Ez. Fitxategiak erreferentziak baino ez ditu gordetzen (HubSpot ataria, Drive karpeta), inoiz ez tokenik edo pasahitzik. Inportatzen duenak bere konektoreak berriz konektatu beharko ditu.

**Zer dauka zehazki?** Proiektuaren metadatuak eta CRMaren ezaugarri bakoitzaren egoera. Ez ditu HubSpoteko datu biziak biltzen, aplikazioak kudeatzen duen konfigurazioa baizik.
