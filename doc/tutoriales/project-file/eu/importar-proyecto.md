# Inportatu proiektu bat fitxategi batetik

**Aurrebaldintzak:** lehenago esportatutako `.rvproj` fitxategi bat izatea.
**Denbora estimatua:** 2 minutu

Proiektu bat zure ekipoan birsor dezakezu `.rvproj` fitxategi batetik.

## Urratsak

1. Ongietorri-pantailan, sakatu **Inportatu proiektua**.
2. Hautatu `.rvproj` fitxategia eta berretsi.
3. Berrikusi **aurrebista-laburpena**: izena, erreferentziatutako konektoreak eta inportatuko den edukia.
4. Identifikatzaile bera duen proiektu bat badago jada, aukeratu **Sortu kopia bat** (lehenetsia) edo **Gainidatzi lehendik dagoena**.
5. Sakatu **Inportatu**.
6. Inportatu ondoren, berriz konektatu **HubSpot** eta **Google Drive** zure kredentzialekin.

## Espero den emaitza

Proiektua zerrendan agertzen da eta irekita geratzen da, bere propietate-mapa, objektu pertsonalizatuak eta inprimakiekin.

## Ohiko galderak

**Zergatik konektatu behar ditut berriz konektoreak?** Fitxategiak ez duelako kredentzialik segurtasunagatik; atariaren eta karpetaren erreferentziak baino ez.

**Zer gertatzen da atal bat nire aplikazioa baino berriagoa bada?** Inportatzean saltatzen da, baina fitxategian gordetzen da, berriz esportatzen baduzu galtzen ez dadin.

**Eta fitxategia aldatuta badago?** Osotasun-egiaztapena bat ez badator, ohar bat ikusiko duzu laburpenean inportatu aurretik.
