# Exportar un proxecto a un arquivo

**Prerrequisitos:** ter un proxecto na lista de benvida.
**Tempo estimado:** 1 minuto

Podes gardar un proxecto completo nun único arquivo `.rvproj` para compartilo ou trasladalo a outro equipo.

## Pasos

1. Na pantalla de benvida, localiza o proxecto na lista.
2. Preme a icona **Exportar** (frecha cara abaixo) na fila do proxecto.
3. Elixe onde gardar o arquivo `.rvproj` e confirma.
4. Verás un aviso de confirmación coa ruta onde se gardou.

## Resultado esperado

Un arquivo `.rvproj` co mapa de propiedades, os obxectos custom e os formularios do proxecto, listo para compartir.

## Preguntas frecuentes

**Inclúe as miñas credenciais?** Non. O arquivo garda só referencias (portal de HubSpot, cartafol de Drive), nunca tokens nin contrasinais. Quen o importe deberá reconectar os seus propios conectores.

**Que contén exactamente?** Os metadatos do proxecto e o estado de cada característica do CRM. Non inclúe os datos vivos de HubSpot, só a configuración que xestiona a app.
