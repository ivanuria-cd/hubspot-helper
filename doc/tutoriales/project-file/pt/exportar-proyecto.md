# Exportar um projeto para um ficheiro

**Pré-requisitos:** ter um projeto na lista de boas-vindas.
**Tempo estimado:** 1 minuto

Pode guardar um projeto completo num único ficheiro `.rvproj` para o partilhar ou transferir para outro equipamento.

## Passos

1. No ecrã de boas-vindas, localize o projeto na lista.
2. Prima o ícone **Exportar** (seta para baixo) na linha do projeto.
3. Escolha onde guardar o ficheiro `.rvproj` e confirme.
4. Verá um aviso de confirmação com o caminho onde foi guardado.

## Resultado esperado

Um ficheiro `.rvproj` com o mapa de propriedades, os objetos custom e os formulários do projeto, pronto para partilhar.

## Perguntas frequentes

**Inclui as minhas credenciais?** Não. O ficheiro guarda apenas referências (portal do HubSpot, pasta do Drive), nunca tokens nem palavras-passe. Quem o importar terá de voltar a ligar os seus próprios conetores.

**O que contém exatamente?** Os metadados do projeto e o estado de cada característica do CRM. Não inclui os dados vivos do HubSpot, apenas a configuração que a aplicação gere.
