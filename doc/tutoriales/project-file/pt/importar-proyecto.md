# Importar um projeto a partir de um ficheiro

**Pré-requisitos:** ter um ficheiro `.rvproj` exportado previamente.
**Tempo estimado:** 2 minutos

Pode recriar um projeto no seu equipamento a partir de um ficheiro `.rvproj`.

## Passos

1. No ecrã de boas-vindas, prima **Importar projeto**.
2. Selecione o ficheiro `.rvproj` e confirme.
3. Reveja o **resumo prévio**: nome, conetores referenciados e conteúdo que será importado.
4. Se já existir um projeto com o mesmo identificador, escolha **Criar cópia** (opção por predefinição) ou **Substituir o existente**.
5. Prima **Importar**.
6. Após importar, volte a ligar o **HubSpot** e o **Google Drive** com as suas credenciais.

## Resultado esperado

O projeto aparece na lista e fica aberto, com o seu mapa de propriedades, objetos custom e formulários.

## Perguntas frequentes

**Porque é que tenho de voltar a ligar os conetores?** Porque o ficheiro não contém credenciais por segurança; apenas as referências ao portal e à pasta.

**O que acontece se uma secção for de uma versão mais recente do que a minha aplicação?** É omitida ao importar mas conserva-se no ficheiro, de modo que não se perde se voltar a exportar.

**E se o ficheiro estiver alterado?** Se a verificação de integridade não bater certo, verá um aviso no resumo antes de importar.
