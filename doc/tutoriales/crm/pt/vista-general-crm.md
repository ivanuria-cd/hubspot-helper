# Vista geral do CRM

**Pré-requisitos:** ter um projeto aberto.
**Tempo estimado:** 1 minuto

O ecrã **CRM** resume as três áreas de trabalho do CRM — Propriedades, Objetos custom e Formulários — num único relance.

## Passos

1. No menu lateral, prima **CRM**.
2. Cada cartão mostra o **total** de elementos da área e quantas **alterações pendentes** tem (ou «Atualizado» se não houver nenhuma).
3. Prima **Abrir** num cartão para ir para essa área e trabalhar nela.
4. Se o HubSpot não estiver ligado, verá um aviso com uma ligação para o configurar (sem portal não há dados de CRM).

## Resultado esperado

Sabe num relance quanto tem em cada área do CRM e onde há trabalho pendente.

## Perguntas frequentes

**Em que se distingue do Dashboard?** O Dashboard cobre todo o projeto (conetores e pendentes). Esta vista centra-se apenas no CRM e acrescenta o total de elementos por área.
