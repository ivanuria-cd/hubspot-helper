# Associer un formulaire à une source

**Prérequis :** formulaires importés (voir « Importer les formulaires existants ») et au moins une source définie dans **CRM → Propriétés**.
**Temps estimé :** 2 minutes

Associer un formulaire à une ou plusieurs sources indique à l'application par rapport à quel ensemble de propriétés elle doit mesurer la couverture du formulaire.

## Étapes

1. Accédez à **CRM → Formulaires**.
2. Cliquez sur le formulaire que vous souhaitez associer pour ouvrir son panneau latéral.
3. Cliquez sur **Associer à une source**.
4. Choisissez l'**objet** de HubSpot par rapport auquel la couverture sera évaluée (par exemple, les contacts).
5. Cochez une ou plusieurs **sources**.
6. Cliquez sur **Enregistrer**.

## Résultat attendu

Le formulaire affiche les sources associées sous forme d'étiquettes, et le panneau calcule le rapport de couverture pour chaque source.

## Questions fréquentes

**Puis-je associer plusieurs sources à un même formulaire ?** Oui. La couverture est calculée séparément pour chaque source.

**Où sont enregistrées les associations ?** Dans l'état local du projet (pas dans HubSpot). Vous pouvez éventuellement les exporter vers Google Sheets.
