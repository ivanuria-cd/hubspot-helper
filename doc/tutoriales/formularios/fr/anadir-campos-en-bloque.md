# Ajouter en masse les champs manquants

**Prérequis :** un formulaire associé à une source avec une couverture incomplète (voir « Examiner la couverture d'un formulaire »).
**Temps estimé :** 3 minutes

Lorsqu'il manque à un formulaire des champs que sa source exige, l'application peut tous les préparer d'un coup sous forme de modification en attente, prête à être examinée et appliquée.

## Étapes

1. Accédez à **CRM → Formulaires** et ouvrez le formulaire incomplet.
2. Dans le rapport de couverture, repérez la source dont des champs manquent.
3. Cliquez sur **Ajouter les champs manquants (N)**.
4. Une **modification en attente** de type « ajouter des champs » est générée ; elle inclut uniquement les champs absents (les champs existants sont conservés).
5. Examinez et appliquez la modification depuis **Modifications en attente** (voir « Synchroniser les modifications avec HubSpot »).

## Résultat attendu

Une modification en attente apparaît avec le résumé « Ajouter N champ(s)… ». Rien n'est écrit dans HubSpot tant que vous ne l'appliquez pas.

## Questions fréquentes

**Les champs sont-ils dupliqués ?** Non. Seuls les champs manquants sont ajoutés ; les champs déjà présents dans le formulaire ne sont pas touchés.

**Avec quelles options sont-ils ajoutés ?** En tant que champs visibles et non obligatoires par défaut. Les réglages fins de style et d'obligation se gèrent ensuite dans HubSpot.
