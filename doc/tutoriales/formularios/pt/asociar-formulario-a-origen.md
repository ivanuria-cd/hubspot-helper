# Associar um formulário a uma origem

**Pré-requisitos:** formulários importados (ver «Importar os formulários existentes») e pelo menos uma origem definida em **CRM → Propriedades**.
**Tempo estimado:** 2 minutos

Associar um formulário a uma ou várias origens diz à aplicação contra que conjunto de propriedades deve medir a cobertura do formulário.

## Passos

1. Entre em **CRM → Formulários**.
2. Clique no formulário que pretende associar para abrir o seu painel lateral.
3. Prima **Associar a origem**.
4. Escolha o **objeto** do HubSpot contra o qual a cobertura será avaliada (por exemplo, contactos).
5. Selecione uma ou várias **origens**.
6. Prima **Guardar**.

## Resultado esperado

O formulário mostra as origens associadas como etiquetas, e o painel calcula o relatório de cobertura para cada origem.

## Perguntas frequentes

**Posso associar várias origens a um mesmo formulário?** Sim. A cobertura é calculada separadamente para cada origem.

**Onde ficam guardadas as associações?** No estado local do projeto (não no HubSpot). Opcionalmente pode exportá-las para o Google Sheets.
